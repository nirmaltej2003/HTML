# HTML Overview

**HTML (HyperText Markup Language)** is the standard markup language used to create **web pages**. It provides the structure of web content by using elements and tags.

## Key Features

- **Markup Language:** Defines the structure and layout of web pages.  
- **Easy to Learn:** Simple syntax with readable tags.  
- **Platform Independent:** Works across all browsers and operating systems.  
- **Supports Multimedia:** Can include images, audio, video, and interactive forms.  
- **Hyperlinks:** Enables navigation between web pages using links.  
- **Semantic Elements:** Tags like `<header>`, `<footer>`, `<article>` provide meaning to content.  

## HTML Structure

A basic HTML document has the following structure:

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <h1>Heading</h1>
    <p>This is a paragraph.</p>
  </body>
</html>
